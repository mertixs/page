四. 指针

pointer是ctypes模块提供的一个方法，可以生成一个指针对象实例(Python)

而POINTER也是一个方法，它的作用是根据传入的类型，制造出新的，你自己定义的数据类型。

自定义的指针类型创建对象需要传入c_type基本类型作为参数，否则会报错。

直接对pointer自定义的指针类型不带参数的初始化会产生NULL空指针。
