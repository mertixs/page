总结

腾讯云机器翻译服务的调用参数分为公共参数和API接口参数

注意：

1. 使用GET和POST请求均可
2. 请求操作需要签名，支持sha1和sha256，签名包含请求方式+请求接口地址+请求参数（&连接）
3. 签名中使用的请求参数需要安装字母ASCII升序排列(A-Z)，参数本身首字母大写，所以不用考虑大小写问题。但实际请求地址无需排序。
