列表生成器

快速产生列表的一种手段

迭代器

通过列表生成式
生成器都是Iterator对象，但list、dict、str虽然是Iterable，却不是Iterator
把list、dict、str等Iterable变成Iterator可以使用iter()函数

可以直接创建满足一定规律的列表，但有时可能只是为了使用其中几个元素，这样就会造成很大的内存浪费，于是就可以可以做简单的改变

这个时候产生的对象就是一个迭代器generator，迭代器的好处就是，它不是返回一个列表对象，这样可以减少内存的开销，每次通过next(g)可以获得下一个返回值，直到抛出StopIteration异常为止

map/reduce

  1. map()

    传入的第一个参数是f，即函数对象本身。由于结果r是一个Iterator，Iterator是惰性序列，因此通过list()函数让它把整个序列都计算出来并返回一个list

  2. reduce()

    reduce把一个函数作用在一个序列[x1, x2, x3, …]上，这个函数必须接收两个参数，reduce把结果继续和序列的下一个元素做累积计算

filter()

和map()类似，filter()也接收一个函数和一个序列。和map()不同的是，filter()把传入的函数依次作用于每个元素，然后根据返回值是True还是False决定保留还是丢弃该元素

python内置属性和方法

__slots__（属性）

在python中，由于其动态的特性，可以添加其他的属性，但如果要限定该类可以添加的属性，可以在原来的类中添加__slots__属性

slots定义的属性仅对当前类实例起作用，对继承的子类是不起作用的

__str__和__repr__
___str__()返回用户看到的字符串，而__repr__()返回程序开发者看到的字符

__getattr__()

当调用不存在的属性时，比如score，Python解释器会试图调用getattr(self, ‘score’)来尝试获得属性

__call__()

一个对象实例可以有自己的属性和方法，当我们调用实例方法时，我们用instance.method()来调用,python中可以用类名()来调用

通过callable()函数，我们就可以判断一个对象是否是“可调用”对象
